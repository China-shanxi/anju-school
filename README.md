# 孝义市安居街小学 静态官网
纯前端校园展示网站，基于HTML/CSS/JavaScript开发，无数据库，可部署GitHub Pages在线访问。

## 网站板块
1. 首页轮播横幅
2. 校园简介
3. 新闻通知公告
4. 教师风采展示
5. 班级园地
6. 校园相册
7. 联系我们

## 特色
1. 官方蓝色校园风格，适配手机、电脑
2. 顶部导航固定悬浮，页面平滑滚动
3. 首页自动轮播大图
4. 响应式布局，手机自动适配
5. 全部静态文件，上传即可部署

## 修改说明
1. 修改学校介绍：index.html 中【校园简介】段落
2. 更新新闻：编辑 news-item 模块文字与日期
3. 修改教师信息：teacher-card 内文字
4. 更换地址电话：联系我们板块
5. 更换轮播、相册图片：替换CSS背景图片地址

## 部署教程
1. 新建GitHub仓库，上传 index.html / style.css / script.js / README.md
2. 仓库 Settings → Pages
3. 分支选择 main，根目录部署
4. 等待1分钟生成在线访问链接
